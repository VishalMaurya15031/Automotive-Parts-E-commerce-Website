# 📌 Project Objective

The objective of this project is to create a scalable and user-friendly platform for purchasing automotive spare parts online.

Customers often struggle to find compatible vehicle parts. This system solves the problem by providing:

- Smart product search
- Category-based browsing
- Secure payment gateway
- Order tracking system
- Admin dashboard for management

# ⚙️ Installation Guide

## 1 Clone repository

git clone https://github.com/VishalMaurya15031/Automotive-Parts-E-commerce-Website.git


## 2 Open project folder

cd Automotive-Parts-E-commerce-Website


## 3 Install dependencies

npm install


## 4 Setup Environment Variables

Create a file named .env.local and add:

NEXT_PUBLIC_RAZORPAY_KEY_ID=your_key_here
RAZORPAY_KEY_SECRET=your_secret_here


## 5 Run project

npm start

# 💳 Payment Integration

This project uses Razorpay Payment Gateway for secure transactions.

Steps:

1. Create Razorpay account
2. Generate API keys
3. Add keys inside .env.local file
4. Use test mode for development
